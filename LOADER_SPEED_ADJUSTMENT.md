# Loading 動畫速度調整

**日期**: 2025-10-01  
**調整**: 放慢數字跳動速度 + 增加字型高度

---

## ✅ 修改內容

### 1. **放慢數字跳動速度** (2倍慢)
```typescript
// Before: minDurationMs = 1200
// After:  minDurationMs = 2400

每個數字停留時間:
Before: 1200ms / 7 = 171ms
After:  2400ms / 7 = 343ms (慢了 2倍!)
```

### 2. **增加字型高度**
```typescript
// Before: lineHeight: 0.85
// After:  lineHeight: 1.1

視覺效果:
- 數字在垂直方向佔據更多空間
- 更有存在感
- 不會感覺被壓扁
```
